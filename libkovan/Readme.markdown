libkovan
========

libkovan is a user library that can read sensor values, move motors, and draw graphics to the Kovan. libkovan is written in C++, but also exposes C bindings for simplified use cases.

Requirements
============

* CMake 2.6.0 or higher
* i2c_wrapper
